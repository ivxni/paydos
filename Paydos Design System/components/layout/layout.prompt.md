Surface + heading primitives.

```jsx
<Card interactive>
  <h3>Doppelapfel</h3>
  <p>The house classic.</p>
</Card>

<Card feature>Empfohlen heute</Card>

<SectionHeading
  eyebrow="Die Karte"
  title="Shisha & spirits"
  subtitle="Hand-packed bowls and a short, considered bar."
/>
```

`Card` ‚Äî frosted glass surface. `interactive` for hover lift + brighter glass, `feature` for a silver-lit highlighted variant, `padding` override. `SectionHeading` ‚Äî `eyebrow`, `title` (renders in chrome silver), `subtitle`, `align`, `accent` to tint the eyebrow purple (use sparingly).
